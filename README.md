# # 検査データ分布比較システム (MVP)

## 🚀 システム概要
CSVファイルをアップロードし、統計的なバリデーションと分析を行うミニマルなツールです。データの正規性チェックや記述統計量の計算が可能です。

## 🛠️ セットアップと実行手順

### 1. 環境構築
必要なライブラリをインストールします。
```bash
pip install -r requirements.txt
```

### 2. Docker Composeでの起動 (推奨)
Docker環境を使用して、依存関係と全てのサービスをまとめて立ち上げます。この方法が最も推奨されます。

```bash
docker-compose up --build
```
`app` サービスが自動的に起動し、Streamlitアプリケーションを利用可能になります。(通常は http://localhost:8501)
### 3. 使用方法
1.  GUI画面から分析対象のCSVファイルをアップロードします。
2.  表示されたカラムリストから、解析したい数値データのみを選択します。
3.  「解析を実行し、分布比較を行う」ボタンをクリックすると、統計結果とQ-Qプロットが表示されます。
